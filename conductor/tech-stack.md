# Tech Stack: E-commerce CLV Predictor

## Backend & Machine Learning
- **Language:** Python 3.10+
- **API Framework:** **FastAPI** – High-performance, asynchronous web framework for building APIs.
- **Deep Learning:** **PyTorch** – Sequential MLP architecture for capturing non-linear relationships.
- **Data Science:** **Pandas** and **Scikit-Learn** – Used for EDA, feature engineering, and baseline modeling (Random Forest).

## Frontend
- **Framework:** **React (Vite)** – Fast, component-driven UI development.
- **Language:** **TypeScript** – Ensures type safety across the frontend codebase.
- **Runtime:** **Bun** – High-performance JavaScript runtime and package manager.

## UI & Styling
- **CSS Framework:** **Tailwind CSS** – Utility-first CSS for responsive, modern design.
- **Component Library:** **Shadcn UI** – Accessible, customizable UI components based on Radix UI.

## DevOps & Orchestration
- **Package Management:** **uv Workspaces** – Fast, reliable Python dependency management.
- **Containerization:** **Docker** – For portable, cloud-agnostic deployment (AWS, Kubernetes).
- **Automation:** **Makefiles** – Centralized orchestration for training, testing, and development tasks.
- **Hosting:** **Vercel** (Frontend) and Cloud-based K8s/ECS (Backend).
